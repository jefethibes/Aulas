Exercício 2

Elaborar um formulário HTML que tenha como entrada três valores para os lados de um triângulo. Escreva uma função (tipoTriangulo) que receba três parâmetros para os lados deste triângulo. Imprima sua classificação (equilátero, isósceles ou escaleno) em uma div. A passagem dos parâmetros deve ser feita de forma simplificada dentro do HTML no evento onclick de um botão ou link da seguinte forma: tipoTriangulo(txtLado1.value, txtLado2.value, txtLado3.value).

Exercício 3

Deseja-se calcular a conta de consumo de energia elétrica de uma casa. Para isso, elabore um formulário em HTML que leia a quantidade de Kwh consumidos e o valor unitário do Kwh. Escreva uma função em JavaScript que faça o cálculo (valor = quantidade x valor unitário) e, caso a quantidade de Kwh ultrapasse 100, o valor do Kwh deve ser acrescido em 25%. Caso ultrapasse 200, o mesmo valor deve ser acrescido em 50%. Os valores devem ser repassados para uma função em JavaScript conforme o exercício anterior.
